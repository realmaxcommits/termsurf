# Issue 744: Update website icon

## Goal

Replace the website logo and favicon with the new TermSurf surfer logo
(`assets/termsurf-11-transparent.png`).

## Background

The website at `website/` uses two icon assets:

- **Logo** (`website/public/logo.png`) — displayed in the header via
  `Header.tsx`. Previously generated by `scripts/resize-logo.ts` from an old
  source image in `termsurf-macos/`.
- **Favicon** (`website/public/favicon.ico`) — the browser tab icon.

A new logo has been designed (`assets/termsurf-11-transparent.png`) — a hooded
surfer on a wave with a transparent background. The logo has already been copied
to `website/public/logo.png` (Experiment 1). The favicon still uses the old
icon.

### References

- `website/src/components/Header.tsx` — `<img src="/logo.png">` (64x64 display)
- `website/scripts/resize-logo.ts` — resize script, currently points to old
  source path (`termsurf-macos/icon-source/termsurf-icon.png`)
- `website/public/favicon.ico` — old favicon

## Experiments

### Experiment 1: Replace website logo

#### Description

Copy the new logo to `website/public/logo.png`, replacing the old one.

#### Changes

```bash
cp assets/termsurf-11-transparent.png website/public/logo.png
```

#### Verification

1. Open the website — new surfer logo appears in the header.

**Result:** Pass

The new logo displays correctly in the header.

#### Conclusion

Logo replaced. Favicon still needs updating.

### Experiment 2: Generate favicon from new logo

#### Description

Generate a new `favicon.ico` from `assets/termsurf-11-transparent.png`. ICO
files typically contain multiple sizes (16x16, 32x32, 48x48). Use `sharp` to
resize and convert, or use ImageMagick/`sips` if available.

Also update `website/scripts/resize-logo.ts` to point at the new source image so
future runs regenerate from the correct file.

#### Changes

**`website/scripts/resize-logo.ts`** — Update the `SOURCE` path:

```typescript
const SOURCE = join(
  import.meta.dir,
  "../../assets/termsurf-11-transparent.png",
);
```

**Generate favicon** — Use the resize script or a one-off command to create
`website/public/favicon.ico` from the new logo at 32x32.

#### Verification

1. Open the website — favicon in the browser tab shows the new surfer logo.
2. `website/scripts/resize-logo.ts` points to the new source image.
